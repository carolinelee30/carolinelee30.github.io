How to add things to the online version of my website

  1. Open terminal and navigate to my folder by typing cd./desktop/carolinelee30.github.io
  2. If I want to see any changes I can type "git status" and it shows changes
  3. To add these changes I type git add.
  4. To commit the changes, I type git commit -m "message here"
  5. To finalize everything, I type git push.
